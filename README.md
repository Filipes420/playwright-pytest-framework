# Playwright Pytest Test Automation Framework

This project is a test automation framework built with Python, Playwright, and Pytest.  
It is designed to automate UI testing of web applications using a scalable and maintainable structure.


## 🧰 Tech Stack

- Python
- Playwright
- Pytest

## 🏗 Project Structure

tests/       -> test cases
pages/       -> POM classes
conftest.py  -> fixtures
